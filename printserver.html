MESA Print Server

Keep this tab open on the laptop — prints all tickets and receipts silently for every device

Setup — do this once on the laptop

1. Go to qz.io → Download → install QZ Tray (free) → it runs in system tray
2. Type your printer name below — exact name from Windows Printers & scanners
3. Click Connect QZ Tray
4. A popup appears → click Allow → tick Remember this decision → done
5. To stop popup forever: right-click the QZ Tray icon in system tray → Advanced → Allow unsigned or set Trust level: Always
